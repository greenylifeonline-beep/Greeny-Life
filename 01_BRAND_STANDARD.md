# Fixed Rules

The GREENY LIFE logo is always green.

Background is always white.

Product layout never changes.

Only product colors change.

The lid color represents the product.

All package sizes keep the same visual identity.

The bottom green strip remains on every label.

QR Code is mandatory.

Batch Number is mandatory.

Country of Origin must appear on every export label.
